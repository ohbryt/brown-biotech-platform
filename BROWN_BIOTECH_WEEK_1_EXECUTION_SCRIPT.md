# Brown Biotech Week 1 Execution Script

## Goal
Lock the team roles, intake flow, and operating language in the first week.

## Day 1 — Decide
- Confirm primary motion: `Paid Brief → peptide-service`
- Confirm supporting lanes: `biostatx`, `genox-site`, `business-pipeline`
- Confirm approval gates: money, legal, contracts, deployment, public claims, medical claims
- Confirm the 10-person team roster

## Day 2 — Map
- Map the intake flow from request to receipt
- Map the owner for each lane
- Map the approval path for high-stakes items
- Map the weekly review rhythm

## Day 3 — Write
- Write the intake questions
- Write the routing rules
- Write the receipt template
- Write the decision log template

## Day 4 — Publish
- Put the team roster in Notion
- Put the team strategy in Notion
- Put the HQ page map in Notion
- Put the 30-day plan in Notion

## Day 5 — Check
- Review one sample intake end to end
- Verify route, owner, approval flag, and next action
- Verify the receipt is shareable and concise
- Verify the language matches the homepage

## Day 6 — Tighten
- Remove ambiguous steps
- Merge duplicate language
- Shorten any long text
- Fix any missing owner fields

## Day 7 — Lock
- Freeze the week 1 operating language
- Confirm what is ready for weekly use
- List what should wait until week 2

## Week 1 output
- One stable intake path
- One stable team roster
- One stable HQ page structure
- One stable receipt format
- One stable weekly review loop
