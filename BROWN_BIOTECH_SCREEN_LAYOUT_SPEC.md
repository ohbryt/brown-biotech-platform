# Brown Biotech Screen Layout Spec

## Purpose
Describe a single-screen visual HQ that makes the business easy to read at a glance.

## Layout
### Top bar
- Brown Biotech title
- primary motion
- current date / review state
- approval indicator

### Left column
#### Intake flow
- Signal
- Qualify
- Brief
- Route
- Review
- Deliver

#### Service lanes
- peptide-service
- biostatx
- genox-site
- business-pipeline

### Center column
#### Active work
- current requests
- owner
- next action
- due date
- approval state

#### Current brief
- one-line brief summary
- deliverable type
- status

### Right column
#### Proof
- publications
- case notes
- research proof
- partner updates

#### Approval gate
- money
- contracts
- deployment
- public claims
- medical claims

### Bottom row
#### Weekly control panel
- wins
- blockers
- risks
- decisions
- next week priorities

## Visual rules
- premium
- clean
- low-noise
- readable in under a minute
- show only what helps intake, routing, delivery, proof, or review

## What not to show
- long paragraphs
- duplicate docs
- equal-weight lanes
- hidden ownership
- anything that obscures the primary motion

## Bottom line
The screen should feel like a control room, not a document archive.
