# Brown Biotech SMB Operating Pack

## Purpose
Translate the Claude small-business pattern into Brown Biotech language:
- ready from day one
- delegate the work
- works with your tools
- built for trust

## 1) SMB-style homepage structure

### Hero
- One brief, one owner, one next action.
- Primary CTA: Request a Paid Brief
- Secondary CTA: Try browser demo
- Support CTA: View Services
- Trust line: Human approval required for high-stakes actions

### Follow-on sections
1. Ready from day one
2. Works with your tools
3. Delegate the work
4. Built for trust
5. Proof and partner motion
6. Request a brief

## 2) Brown Biotech equivalents

### Ready from day one
- Notion HQ
- service hub
- intake routing
- browser demo
- receipt flow

### Delegate the work
- route every request
- assign one owner
- produce one next action
- keep human review on sensitive items

### Works with your tools
- Notion
- Telegram
- GitHub
- Vercel
- browser test
- research outputs

### Built for trust
- inspectable workflow
- decision-ready handoff
- privacy-aware execution
- no blind automation
- human approval gate

## 3) What to show in the UI
- a visible offer ladder
- a route preview
- a proof layer
- a human approval layer
- a one-page control room

## 4) Notion mapping
Use the same structure in HQ:
- Service Intake Templates
- Intake Requests
- Intake Routing
- Funnel
- Weekly Reviews
- Decision Log
- Research Proof
- Partner Funnel
- 30-Day Plan

## 5) Worker mapping
- intake-triage
- weekly-review
- dataset-signal
- launch-check
- partner-routing

## 6) Connector mapping
- Notion HQ
- GitHub
- Telegram
- Vercel
- Google Drive / PDFs
- PubMed / PMC
- GEO / SRA
- Hugging Face Datasets

## 7) Decision rule
If a page is confusing, simplify it until it can be explained as:
- what comes in
- who owns it
- what happens next
- where the proof goes
- when human review is required

## 8) Success criteria
- homepage looks like an operating system, not a brochure
- service hub is one clear ladder
- Notion mirrors the same language
- browser test shows route preview and receipt
- every high-stakes action is review-gated
